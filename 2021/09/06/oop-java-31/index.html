OOP - Design Pattern - Facade

Posted on 2021-09-06 In java , oop , design pattern Views:

Facade Pattern

  • 구조 패턴
  • 정면, 앞면이라는 뜻을 가지는 Facade는 여러 개의 객체와 실제 사용하는 서브 객체 사이에 복잡한 의존관계가 있을때 중간에 Facade 객체를 두어 제공하는 interface만을 활용하여 기능을 제공하는 방식이다
  • Facade는 자신이 가지고 있는 각 클래스의 기능을 명확히 알아야 한다

예제

  • 해당 클래스를 클라이언트가 사용하기 위해서는
  • Client가 의존성을 가지고있는 형태가 된다
  • SftpClient를 만듬으로써 의존성을 가져간다
  • 여러 객체의 의존성을 뒤로 숨겨주어 기능적으론 동일하지만 간략하게 줄일 수 있다.
