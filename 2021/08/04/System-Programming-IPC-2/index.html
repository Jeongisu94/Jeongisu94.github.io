커널 영역에 메시지큐나 공유메모리가 생성이되면 명시적으로 컨트롤 해주는 것이 필요하다

공유메모리 (Shared Memory)

  • 모든 프로세스는 커널영역을 공유한다는 점을 이용하여
    커널에 공간을 확보하여 그 공간을 변수처럼 사용하는 방식이다
  • 큐와 다르게 주소를 변수처럼 접근한다
  • 공유 메모리 Key를 통하여 여러 프로세스가 접근이 가능하다
