    CSS 기초

    Posted on 2021-07-21 Edited on 2021-07-30 In CSS 
    div {
	font-size: 20px;
	color: red;
}

    선택자 안에 속성과 값을 정의한다.

    선택자 (selector)

    선택자는 HTML에 CSS를 적용하기 위한 HTML의 특정한 요소를 선택하여 명시해주는 역할을 한다.

    <div>
	<h1>Title</h1>
	<p>Content</p>
</div>
    h1 {
	color: red;
}
p {
	color: blue;
}

    다음과 같이 사용할 수 있다.

    클래스 선택자

    /* class.css */
/* class="title" */
.title {
	color: red;
}
.main-text {
	color: green;
}
    <!-- html file  -->
<h1 class="title"> title </h1> 
<p class="main-text"> main text </p>

    . 을 이용해서 클래스임을 나타낸다.

    선택자 없이 인라인 방식으로 직접 작성 할 수있다

    <div style="color: red;">Content</div>

    직접 손으로 작성할 때 코드가 길어질 수록 하드코딩 해야하고 수정시 굉장히 번거로워 질 수있어서 지양해야한다.

    HTML에 Embed하기

    <head>
	<style>
		div {
			속성: 값;
		}
	</style>
</head>
<body>
	<div>content 1</div>
	<div>content 2</div>
	<div>content 3</div>
</body>

    HTML 외부에서 불러오기 ( link 태그 이용)

    <head>
	<link rel="stylesheet" href="/css/main.css"> 
</head>
    /* main.css */
div {
	color :red;
}

    rel은 현재 HTML의 문서와 불러오는 문서와의 관계를 나타내며 href는 그 문서의 경로이다.

    CSS를 외부에서 불러올 경우 재사용에 용이하다.

    속성(Properties)과 값(Value)

    div {
	속성1: 값;
	속성2: 값;
	...
}
